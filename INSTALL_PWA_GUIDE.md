# 📱 Nutri8 앱 설치 가이드

## 🎯 웹사이트를 앱처럼 사용하기

Nutri8 웹사이트를 폰 홈 화면에 앱 아이콘으로 추가할 수 있습니다!

**장점:**
- ✅ 브라우저 주소창 없이 전체 화면으로 실행
- ✅ 홈 화면에서 바로 실행 (URL 입력 불필요)
- ✅ 앱처럼 빠른 실행
- ✅ 멀티태스킹에서 앱으로 표시

---

## 📱 Android 설치 방법

### **방법 1: 자동 설치 배너 (권장)**

1. **Chrome 또는 Samsung Internet**에서 접속:
   ```
   https://nutri8-6z1o.vercel.app/
   ```

2. **설치 배너 대기** (5-10초):
   ```
   ┌─────────────────────────────────┐
   │ Nutri8을(를) 설치하시겠습니까?    │
   │                                 │
   │  [설치]  [나중에]                │
   └─────────────────────────────────┘
   ```

3. **[설치]** 버튼 클릭

4. **완료!** 🎉
   - 홈 화면에 Nutri8 아이콘이 생성됩니다
   - 아이콘을 클릭하면 앱처럼 실행됩니다

---

### **방법 2: 수동 설치**

1. **웹사이트 접속**:
   ```
   https://nutri8-6z1o.vercel.app/
   ```

2. **Chrome 메뉴 열기**:
   - 우측 상단 ⋮ (점 3개) 클릭

3. **"홈 화면에 추가" 선택**:
   ```
   ⋮ 메뉴
   ├─ 새 탭
   ├─ 새 시크릿 탭
   ├─ 북마크
   ├─ 홈 화면에 추가 ← 여기!
   └─ 설정
   ```

4. **"추가" 버튼 클릭**:
   ```
   ┌─────────────────────────────┐
   │  홈 화면에 추가               │
   ├─────────────────────────────┤
   │  이름: Nutri8                │
   │                             │
   │  [취소]  [추가] ← 클릭!      │
   └─────────────────────────────┘
   ```

5. **완료!** 🎉

---

## 🍎 iOS (iPhone/iPad) 설치 방법

### **Safari 브라우저 사용 (필수)**

**⚠️ 중요: iOS에서는 Safari만 PWA 설치를 지원합니다!**

1. **Safari에서 접속**:
   ```
   https://nutri8-6z1o.vercel.app/
   ```

2. **공유 버튼 클릭**:
   - 하단 중앙의 📤 (공유) 버튼 클릭
   ```
   ┌─────────────────────────────┐
   │  Safari 주소창               │
   │  https://nutri8...          │
   │                             │
   │  ← ⬆️ → 🔖 📄             │
   │       ↑ 여기 클릭!           │
   └─────────────────────────────┘
   ```

3. **"홈 화면에 추가" 선택**:
   - 아래로 스크롤하여 찾기
   ```
   공유 메뉴
   ├─ AirDrop
   ├─ 메시지
   ├─ 메일
   ├─ ...
   ├─ 📱 홈 화면에 추가 ← 여기!
   └─ ...
   ```

4. **앱 이름 확인 및 "추가" 클릭**:
   ```
   ┌─────────────────────────────┐
   │  🍎                         │
   │  Nutri8                     │
   │  nutri8-6z1o.vercel.app     │
   │                             │
   │  [추가] ← 클릭!              │
   └─────────────────────────────┘
   ```

5. **완료!** 🎉
   - 홈 화면에 Nutri8 아이콘이 생성됩니다

## ⚠️ 문제 해결

### **"홈 화면에 추가" 옵션이 안 보여요**

**Android:**
- Chrome 또는 Samsung Internet 사용 확인
- 최신 버전으로 업데이트
- 시크릿 모드가 아닌지 확인

**iOS:**
- Safari 브라우저 사용 (Chrome/Firefox 불가)
- iOS 11.3 이상 버전 확인

---

### **설치 배너가 안 나타나요**

**해결 방법:**
- 방법 2 (수동 설치) 사용
- 브라우저 메뉴 → "홈 화면에 추가"

---

### **아이콘이 이상해요**

**확인 사항:**
- 배포 완료 대기 (1-2분)
- 브라우저 캐시 삭제
- 하드 리프레시: Ctrl + Shift + R
